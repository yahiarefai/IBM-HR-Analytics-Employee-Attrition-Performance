# Employee Attrition Prediction

## Overview
This project analyzes employee attrition using machine learning techniques to identify employees at risk of leaving.

## Objectives
- Understand key factors influencing employee attrition
- Build a predictive model

## Key Insights
- Lower monthly income increases attrition risk
- Overtime significantly impacts employee turnover
- Job satisfaction is critical for retention

## Machine Learning
- Model: Random Forest Classifier
- Handled imbalanced data using class weighting
- Improved recall using threshold tuning

## Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Scikit-learn
